# MeetingClone

Lightweight Next.js demo: one-to-many WebRTC meeting UI with Firebase Realtime Database signaling.

## Quick start

Prerequisites
- Node.js 18+
- A Firebase Realtime Database (fill config in [src/lib/firebase.ts](src/lib/firebase.ts))

Install & run
```sh
npm install
npm run dev
# open http://localhost:3000
```
